# 핫개수 :  핫한 개들의 수다
> 본 프로젝트는 반려인들을 위한 SNS형 커뮤니티 플랫폼입니다.

Spring MVC패턴과 REST API로 구현

![](../header.png)

## 기술 스텍

* Back-End : Spring | JAVA | Python | MyBatis
* Front-End : HTML5 | CSS | JavaScript(JQuery) | BootStrap
* DB : MySQL
* Tool : STS3
* API : KAKAO MAP | NAVER LOGIN | KAKAO LOGIN | OpenWeatherMap
